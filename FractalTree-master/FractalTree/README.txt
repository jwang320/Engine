The main project for simulation is FractalTree32, it includes the compiled code for the following libraries:

Rendering Engine Base:
	A static library to set up basic directx functionality (create a window with 3d, etc)

Rendering Engine Standard:
	A static library which assumes the general structure of the engine

Rendering Engine Sample:
	A static library, fully-functioning implementation of the graphics renderer that can be used by any project.


Simulation Engine Base:
	A static library, handles managing an object system

Simulation Engine Physical:
	A static library, simulates objects in a physical world.


SimpleUI:
	A static library, a simple UI.
